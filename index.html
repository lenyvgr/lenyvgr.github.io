03 — Réalisations

Projets sélectionnés

Projet 01

Déploiement d'un cluster de virtualisation Proxmox VE

Proxmox VE HP ProLiant KVM LVM

Mise en place d'un cluster Proxmox VE multi-nœuds sur serveurs HP ProLiant DL360 Gen9. Configuration du stockage partagé, création de templates de VMs et conteneurs LXC, gestion des ressources et des pools.

  • Installation et configuration du cluster avec Corosync
  • Provisionnement de VMs Windows Server 2022 et Debian
  • Sauvegardes automatisées via Proxmox Backup Server
  • Stockage RAID 5 via Intel VROC sur les nœuds
Rôle : Administrateur infrastructure
Projet 02

Déploiement et intégration de HashiCorp Vault

Vault Secrets Mgmt Linux TLS

Installation et configuration de HashiCorp Vault sur un conteneur LXC Debian dédié. Gestion centralisée des secrets et des certificats pour les services internes du cluster.

  • Déploiement Vault en mode standalone avec stockage fichier
  • Configuration des politiques d'accès ACL par service
  • Intégration des secrets dans les VMs du cluster
  • Chiffrement des données sensibles au repos
Rôle : Administrateur systèmes & sécurité
Projet 03

Supervision d'infrastructure avec tableau de bord centralisé

Monitoring SNMPv3 Dashboard Proxmox API

Mise en place d'une solution de supervision pour l'ensemble des serveurs et équipements réseau. Centralisation des métriques (CPU, RAM, stockage, dispo) dans un tableau de bord temps réel.

  • Collecte métriques via API Proxmox et SNMPv3
  • Dashboards par environnement (hyperviseurs, switches, routeurs)
  • Configuration des alertes et seuils de notification
  • Documentation des indicateurs clés et procédures d'escalade
Rôle : Administrateur systèmes & réseaux
Projet 04

Intégration d'une baie serveur dans l'infrastructure réseau

Cisco IOS Enterasys VLAN RSTP UniFi

Configuration et intégration d'un switch manageable pour raccorder une baie de serveurs à l'infrastructure existante. Segmentation VLAN, spanning-tree et sécurisation des ports.

  • VLANs (serveurs, administration, iLO/BMC) et ports trunk
  • RSTP configuré pour éliminer les boucles réseau
  • Port Security et désactivation des ports inutilisés
  • Intégration dans la topologie UniFi et documentation câblage
Rôle : Technicien réseau & infrastructure
